# Multivers Explorer - TP React

## Installation
1. `npm install`
2. `npm run dev`

## Fonctionnalités
- Liste des personnages avec pagination (Précédent/Suivant).
- Page de détail par personnage.
- Formulaire d'évaluation avec validation Zod & Formik.
- Affichage des résultats dans une modale native `<dialog>`.
